# QUESTÕES DE NIVELAMENTO 


## O que é Git? 
````markdown 
- Uma linguagem de programação
- Um sistema de controle de versão local
- Um banco de dados
- Um sistema de controle de versão remoto
````

## Qual comando inicializa um novo repositório Git em uma pasta?
````markdown  
- git start
- git new
- git init
- git create
````
## O que é um repositório git? 
````markdown 
- Um diretório que contém apenas arquivos de código
- Um local que armazena arquivos e o histórico completo de versões
- Um servidor remoto.
- Um tipo de branch principal
````

## Para que serve o GitHub?
````markdown 
- Executar código
- Hospedar repositórios e colaborar em projetos
- Substituir o Git
- Criar banco de dados
````

## Qual a diferença entre Git e GitHub?
````markdown 
- Git é online e GitHub é offline
- Git é um sistema de versionamento, GitHub é uma plataforma de hospedagem
- Não existe diferença
- GitHub substitui o Git
````

## Qual afirmação descreve corretamente a relação entre Git e GitHub?
````markdown 
- Ambos são ferramentas locais de versionamento
- Git depende obrigatoriamente do GitHub para funcionar
- Git é o sistema de versionamento e o GitHub é uma plataforma que o utiliza para hospedagem e colaboração
- GitHub substitui completamente o Git
````

## O que é um commit?
````markdown 
- Um erro no código
- Um envio de alterações com histórico
- Um tipo de branch
- Um arquivo
````
